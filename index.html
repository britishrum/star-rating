Rating control

Rating control allows users select the number of stars that represents their rating.

The control is based on webcomponents technology, so until it is supported by all major browsers it is highly recommended that you use Web Components polyfill. You can either download it manually or build your control from the source code. If you build the control from the sources, remember to install npm and bower dependencies.

Frst, download star and rating components and put it in your project directory, wherever you want them to be. Then, import x-rating component using HTML imports:
<link rel="import" href="rating/rating.html">
Then, put x-rating element on the page like this:
<x-rating></x-rating>
Result:
You can manually set value and amount of stars by setting value and maxvalue attributes:
<x-rating value="7" maxvalue="9"></x-rating>
Result:
To change size of the control, use starsize attribute:
<x-rating starsize="20px"></x-rating>
Result:
<x-rating starsize="80px"></x-rating>
Result:
To change opacity of the control, use opacity attribute:
<x-rating opacity="0.2"></x-rating>
Result:
If you want to prevent user from changing value, you can use disabled attribute:
<x-rating value="4" disabled="true"></x-rating>
Result:
